# 구글 드라이브 공유 학습 시뮬레이터

초등 4~6학년 학생들이 **구글 드라이브에서 문서를 친구와 공유하는 방법**을 안전하게 연습하는
GUI 시뮬레이터입니다. 학급 웹툰 출판 활동(웹툰 PDF, 예: `2-5.pdf`)을 배경으로,
실제 구글 계정·인터넷 없이 동작하는 **단일 HTML 파일**로 만들어졌습니다.

- 진짜 구글 드라이브가 아니며, 무엇을 눌러도 실제로 공유되지 않습니다.
- 등장인물·이메일은 전부 가상(`@haneul.example`)이고, 저장 기능이 없어 새로고침하면 초기화됩니다.

## 파일 구성

| 파일 | 용도 |
|---|---|
| `구글드라이브-공유연습.html` | **원본.** 학생들에게 파일로 배포할 때 이 파일 하나만 전달 |
| `index.html` | 원본의 복사본. 웹 주소로 배포할 때 서버가 기본으로 여는 파일 |
| `.claude/launch.json` | 개발 중 미리보기용 로컬 서버 설정 (배포와 무관) |

> ⚠️ **두 HTML 파일은 항상 같은 내용이어야 합니다.**
> 시뮬레이터를 수정하면 `구글드라이브-공유연습.html`을 고친 뒤 `index.html`로 복사해 주세요.
> (2026-07-12 동기화 완료 — 현재 두 파일 내용 동일)

## 학습 흐름

1. **시작 화면** — 이름 입력(수료증에 사용)
2. **1단계 가이드** — 🦉 도우미 부엉이가 스포트라이트로 클릭할 곳만 밝혀 주며 안내
   - 레슨 ① 친구에게 공유: 파일 선택 → 공유 버튼 → '지우' 입력·자동완성 → 뷰어 권한 → 보내기
   - 레슨 ② 링크 공유: 일반 액세스 변경 → 개인정보 주의 약속 → 링크 복사
   - 레슨 ③ 우클릭으로 공유 창 열기 → 권한 변경 → 액세스 삭제(공유 취소)
3. **권한 퀴즈 4문항** — 뷰어/댓글 작성자/편집자 개념, 링크 공유의 위험
4. **2단계 미션 4개** — 안내 없이 혼자 수행, 막히면 💡힌트 버튼
5. **수료증** — 이름·날짜·퀴즈 점수, 🖨️ 인쇄 버튼(수료증만 인쇄됨)

## 배포 방법

- **파일 배포**: `구글드라이브-공유연습.html`을 구글 드라이브·USB·학급 홈페이지 등으로 전달.
  학생이 더블클릭해 브라우저로 열면 바로 동작 (인터넷 불필요)
- **웹 배포**: 폴더째 웹 호스팅에 올리면 주소만으로 접속 가능 (`index.html`이 기본으로 열림)

## 수정 이력 (2026-07-12)

| 문제 | 수정 |
|---|---|
| 권한 드롭다운 메뉴가 스포트라이트에 가려 클릭 불가 | `position:fixed` 요소의 위치 계산 방식 수정 |
| 가이드 중 어두운 영역(방패막)을 클릭하면 우클릭 메뉴·권한 메뉴가 닫혀 진행 불가 | ① 방패막 클릭으로는 메뉴가 닫히지 않게 ② 메뉴가 사라지면 이전 단계로 자동 복귀 |
| 가이드 중 오답 선택지(예: 편집자)가 눌려서 진행이 막힘 | 오답 잠금 순서 버그 수정 — 이제 정답 선택지만 눌리고 나머지는 흐리게 잠김 |
| 작은 창에서 말풍선이 메뉴 선택지를 가림 | 아래·위 공간이 없으면 말풍선을 대상 옆으로 배치 |

모든 수정은 브라우저에서 가이드 → 퀴즈 → 미션 → 수료증 전 과정을 직접 클릭하며 검증 완료
(콘솔 에러 0건, 태블릿 768px 레이아웃 확인).

## 수정하고 싶을 때 (참고)

- 반 친구 이름/이메일: HTML 내 `PEOPLE` 배열
- 미션 내용: `MISSIONS` 배열
- 퀴즈 문항: `QUIZ` 배열
- 가이드 대사: `GUIDE_STEPS` 배열
